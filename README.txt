== Description ==

Dispay a sticky bar at the bottom or top of posts, pages or products that shows category, post title, author, time needed to read article, share buttons and previous/next post links.

Available settings:

- Choose position: top or bottom of the screen
- Enable/Disable features: Category, Title, Author, Date, Time to Read, Comments, Sharing Buttons, Prev/Next Links
- Select on which Post Types to show: Posts, Pages, WooCommerce Products, Custom Post Types..
- Enable Dark Mode üòé
- Set Color Scheme for the Category and Progress Indicator bar
- Translate "Time to read" and "By" text
- Set average words per minute to calculate the time to read
